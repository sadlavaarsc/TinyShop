## ж ёеҝғзү№жҖ§

- **з§’жқҖй“ҫи·Ҝ**: Sentinel д»ӨзүҢжЎ¶йҷҗжөҒ + Redis Lua еҺҹеӯҗжүЈеҮҸ + RocketMQ ејӮжӯҘдёӢеҚ• + MySQL д№җи§Ӯй”Ғ
- **еҲҶеә“еҲҶиЎЁ**: ShardingSphere жҢүз”ЁжҲ· ID еҲҶзүҮ
- **еҲҶеёғејҸдәӢеҠЎ**: Seata AT жЁЎејҸ
- **зЁіе®ҡжҖ§**: Sentinel зғӯзӮ№йҷҗжөҒ + ж…ўи°ғз”ЁйҷҚзә§ + зҶ”ж–ӯ
